Pok√©dex is a mobile application built with [React Native](https://reactnative.dev/) and [Typescript](https://www.typescriptlang.org/) capable of listing Pok√©mons from various generations and display informations about each of them, just like the Pok√©dex from the original games and anime series. The application was made to practice React Native, consuming third-party APIs, displaying big amount of data with "infinite" scroll, navigation between screens, mobile styling with [Styled Components](https://styled-components.com/) and animations.
